📹

1. Input

Accept video files (MP4) or single images (JPG/PNG) from any source.

OpenCV PIL
🔍

2. Preprocessing

Extract frames, detect faces, crop with padding, resize to 299×299.

MTCNN torchvision
🧠

3. Model

Extract features with XceptionNet or EfficientNet-B0, classify as real/fake.

PyTorch Transfer Learning
📊

4. Aggregation

Combine frame-level predictions into a single video-level decision.

Mean Majority Vote
✅

5. Output

Return prediction (Real/Fake) with confidence score and explanation.

JSON GradCAM
